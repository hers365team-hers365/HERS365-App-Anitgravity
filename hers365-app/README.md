# H.E.R.S.365 — Official Prototype

This repository contains the complete NFL-ready prototype for the H.E.R.S.365 Platform. Designed with a premium dark-mode, glassmorphism aesthetic, advanced UI/UX animations, and a massive 36-table unified database schema.

## Core Features Implemented
- **Immersive Feed & Reels:** Instagram/TikTok style discovery pages.
- **Top 250 Leaderboard:** Live scout rankings, G5 ratings, breakout projections.
- **Athlete Profiles:** In-depth combine stats, verified badges, highlight carousels.
- **NIL Central:** Interactive GPT-4 Advisor Chatbot, Daily Brand Challenges.
- **Training & Drills:** AI-generated personalized workout routines and drill browsing.
- **Recruiting Dashboard:** Profile views, offer tracking, and "Premium" paywall up-sells.
- **Teams & Events:** Live live-leaderboard events, ID camps, and team rosters.
- **Security & Monetization:** Ready-made Stripe Checkout subscriptions and Auth flows.

## Tech Stack
- Frontend: React + TypeScript + Vite + Tailwind CSS (v3) + Framer Motion + Lucide React
- Backend: Node.js + Express + Drizzle ORM + Postgres + OpenAI (GPT-4o)

## Getting Started

1. **Start the Backend Server:**
   ```bash
   cd server
   npm install
   npm run dev
   ```

2. **Start the Frontend Client:**
   ```bash
   cd client
   npm install
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser to experience the platform.
